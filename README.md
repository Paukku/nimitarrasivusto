# nimitarrasivusto
Nimitarrasivusto, jossa käyttäjä voi luoda oman nimitarran. 

Tämä on tarkoitus olla pieni projekti, jossa hyödynnetään kahta eri tekoälyä. Clauden avulla tehdään vaatimusmäärittely, tekninen arkkitehtuuri ja tehtävälista teknisen arkkitehtuurin perusteella. Koodauksessa apuna käytetään Copilottia.

Teknologiat: Node.js, Express.js, TypeScript, React, ReactQuery, PostgreSQL (aluksi in-memory tietokanta)
